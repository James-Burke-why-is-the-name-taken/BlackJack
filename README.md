# BlackJack
When program is run, print paragraph explaining the rules of Blackjack, then prompts them if they want to play
Player is given $100 to bet, as well as 2 random cards. Player can choose how much they want to bet depending on the amount of money they have
Player's cards and dealer's cards are displayed in ASCII art in the console. There cannot be duplicate cards. 

After each match, player is given option to cash out, therefore ending the whole game and displaying their money

Non-main files would have a .h and .c variant, where the .h file declares functions and variables, but the .c file defines them.
Then, the .h file is included in the main.c file, but the .c variant files would be linked when they're all compiled
